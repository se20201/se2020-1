三角函数测试工作
=
编写软件：python、matlab
--
* 在test分支中，对每个三角函数模块进行了1000次求值测试，并将测试的结果逐条地显示在界面下方。要求和系统三角函数的误差小于阈值，若误差大于阈值，  则额外弹出错误消息提示框。
* 在test分支完成测试工作后，将稳定的三角函数模块整合到master分支的最终界面中，并生成2.0版本。  
* 最终界面取消了测试界面的测试结果显示文本框，保留了其Test按钮和测试功能，测试结果改为弹窗提示。
